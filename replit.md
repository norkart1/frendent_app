# Calculator App

## Overview

A simple, sleek, and functional calculator application built with React Native and Expo. It features a dark-themed UI with support for basic arithmetic operations, percentages, and parentheses.

## User Preferences

Preferred communication style: Simple, everyday language.

## System Architecture

### Frontend Framework
- **Technology**: React Native with Expo SDK 52
- **Routing**: Expo Router with file-based routing
- **Rationale**: Expo provides a streamlined development experience, while expo-router offers easy navigation.

### Styling Architecture
- **Pattern**: StyleSheet-based styling inlined in the main screen for simplicity.
- **Theme**: Dark theme with blue and teal accents.

## Features
- Basic arithmetic (+, -, x, รท)
- Parentheses support
- Percentage calculation
- Number formatting with commas
- Clear entry (C)
- Sleek dark mode design
